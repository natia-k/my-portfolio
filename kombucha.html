CONCEPT

ONE FLAVOR. ONE COLOR. NO DECORATION.

Everything is stripped so the fermented color becomes the brand language itself. This project is a deliberate reverse of category expectations — no kraft, no hand-drawn botanicals.

THE SYSTEM

PREMIUM WITHOUT PERFORMANCE.

The palette is drawn directly from fermentation — raw amber, hibiscus blush, blackcurrant near-black. Not colors about the flavors — colors from them.
